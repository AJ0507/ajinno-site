
# AJINNO Studio 簡易官網

這是 AJINNO Studio 的初版靜態網站，內容包括 SAIR 與 AiviA 專案介紹。

## 🚀 快速部署至 Vercel

1. 下載並解壓此 ZIP
2. 前往 https://vercel.com 並使用 GitHub 登入
3. 建立新專案，選擇 "Import Project" → "Add from Git" 或拖入此專案
4. 設定 Project root 為當前資料夾，build 命令與輸出目錄留空即可
5. 綁定自訂網域 `ajinno.com`（需設定 DNS A 記錄或 CNAME）
